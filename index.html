Dans ces exercices, comme en javascript, l'opérateur '+' correspond à l'addition ET la concaténation.
Le signe ← signifie "prend la valeur".

Exercice 1

Variables : A, B, C (entiers)

Début

A ← 3
B ← 2
C ← A + B
A ← B + C
B ← A - C

Fin

A
B
C

Exercice 3

Variables : A, B, C (entiers)

Début

Exercice 4

La boucle va de 1 à A inclus.

Variables : A, B (entiers)

Début

A ← 3
B ← 2
Pour i de 1 jusqu'à A avec un pas de 1 B ← B * i

Fin

B

Exercice 5

Exercice 6

Variables : A (chaîne de caractères) B, C (entiers) D (tableau de caractères)

A ← ""

C ← 0
D ← ['a', 'b', 'c', 'd', 'e', 'f']
Tant que B >= 0 Ajouter à A le caractère de D qui à pour indice B
Si C est paire Décrémenter B de 1 Incrémenter C de 1

Fin
